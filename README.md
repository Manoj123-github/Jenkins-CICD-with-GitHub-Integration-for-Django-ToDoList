# Jenkins-CICD-with-GitHub-Integration-for-Django-ToDoList

Steps:
 1. Created AWS EC2 instance
 2. installed Jenkins and Java dependecies 
 3. Installed Docker
 4. Created a new project on jenkins and integrated with github source code
